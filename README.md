# SkillPath Learning App

SkillPath is a Flutter-based learning platform that helps students improve their technical skills through curated resources and interview preparation.

## Features
- Course-based learning (Python, C++, Java, Flutter, AWS)
- Interview question bank
- Aptitude practice
- Video learning resources
- Progress tracking

## Technologies Used
- Flutter
- Dart
- Web Deployment (Netlify)
